# 项目介绍

本项目为南京大学软件学院人机交互项目——仿豆瓣的书籍信息、评论汇总安卓App

emmm，看了其他组的项目，觉得其实完成度并不是很高，很多同学是很富有技术力和想法的。

时间仓促，并没有来得及配备后端，基本所有数据都是我自己手敲进去的，用Sqlite数据库暂时顶着进行读写操作。
